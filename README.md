# My Profile Site

머신러닝 엔지니어 & 데이터 사이언티스트 개인 프로필 사이트 템플릿입니다.

## 미리보기

- **이름**: Kwang Myung Yu
- **분야**: Machine Learning, Data Science, Automatic Control
- **언어 지원**: 영어/한국어 전환 가능

## 기술 스택

- **HTML5** - 시맨틱 마크업
- **Tailwind CSS** - 유틸리티 기반 스타일링 (CDN)
- **Vanilla JavaScript** - 프레임워크 없이 순수 JS
- **Google Fonts** - Inter, Noto Sans KR

## 프로젝트 구조

```
my-profile-site/
├── index.html          # 메인 HTML 파일
├── css/
│   └── style.css       # 커스텀 스타일
├── js/
│   └── main.js         # JavaScript 로직
├── img/                # 이미지 리소스
│   ├── icon.png        # 프로필 아이콘
│   └── *.svg           # 프로젝트 이미지
└── resume/             # 이력서 파일
```

## 주요 기능

- **반응형 디자인** - 모바일, 태블릿, 데스크톱 지원
- **다국어 지원** - 영어/한국어 전환 버튼
- **스크롤 애니메이션** - 섹션별 페이드인 효과
- **네비게이션** - 고정 헤더, 모바일 메뉴
- **섹션 구성**
  - Hero (소개)
  - About (자기소개)
  - Skills (기술 스택)
  - Projects (프로젝트)
  - Contact (연락처)

## 로컬 실행

별도의 빌드 과정 없이 `index.html`을 브라우저에서 열면 됩니다.

```bash
# VS Code Live Server 사용 시
# 또는 간단한 HTTP 서버 실행
python -m http.server 8000
```
